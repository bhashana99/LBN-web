# Vacancy Manager Pro

Vacancy Manager Pro is a full-stack recruitment management platform developed for **LBN Key Resource (Pvt) Ltd**, an authorized employment agency approved by the Sri Lanka Bureau of Foreign Employment.

The system provides an efficient way to manage job vacancies through a modern admin dashboard, secure authentication, and a responsive user interface for both recruiters and job seekers.

## Project Overview

Vacancy Manager Pro was built as a production-ready recruitment platform to streamline vacancy publishing and management.  
It supports secure login, vacancy administration, and media handling for documents such as PDFs and images.

This project demonstrates real-world full-stack development experience, including authentication, deployment, and scalable architecture.

---

## Key Features

- Admin dashboard for managing job vacancies  
- Secure authentication using JWT  
- Media upload support (PDFs, images) via Firebase Storage  
- Responsive and modern UI with Tailwind CSS  
- Seamless experience for recruiters and job seekers  

---

## Tech Stack

- **Frontend:** React.js, Redux Toolkit, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)  
- **Storage:** Firebase (Images & PDF uploads)  
- **Deployment:** Live production hosting
